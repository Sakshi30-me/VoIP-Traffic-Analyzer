The project **Tracing VoIP via Network Metadata** focuses on analyzing Voice over Internet Protocol (VoIP) traffic to identify the origin, destination, and characteristics of voice calls transmitted over a network. With the increasing use of VoIP technology in both personal and corporate communication, monitoring and securing such calls has become essential to detect potential cyber threats, scams, and unauthorized access attempts.

This project aims to develop a **Python-based system** capable of reading and analyzing packet capture files (PCAP/PKSZ) to extract valuable metadata such as **caller and receiver IP addresses, call duration, timestamp, and geolocation details.** The system leverages the Scapy library for packet inspection and the GeoIP2 API for IP-based location tracking. Using this metadata, the system provides insights into network-level call tracing, making it possible to detect suspicious VoIP activity without relying on tools like Wireshark.

The output of this project is a simple and interactive dashboard or command-line interface that displays call paths and related metadata, assisting cybersecurity professionals and students in understanding how VoIP packets travel across networks. The project also demonstrates **the integration of network analysis, data visualization, and security monitoring,** highlighting a real-world application in VoIP forensics and cyber threat analysis.
