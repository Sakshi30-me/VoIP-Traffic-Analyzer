# 📂 Sample Data

This directory contains small sample `.pcap` files for testing the VoIP Traffic Analyzer.

## ⚠️ Note

* These files are **for demonstration purposes only**
* Depending on the capture content, the tool may:

  * Detect limited or no SIP sessions
  * Show incomplete RTP flows
  * Produce minimal metrics

This is expected behavior, as not all PCAPs contain valid or complete VoIP traffic.

## 🧪 If Sample Files Don’t Work

If no results are shown:

* Try uploading your own PCAP file with VoIP traffic
* Use tools like Wireshark to capture SIP/RTP packets
* Ensure the capture includes active VoIP communication

## 🔒 Disclaimer

Do not upload sensitive or real network traffic without proper authorization.
